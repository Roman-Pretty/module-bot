    Welcome to ECS421U

    This is an introductory module in automata and formal languages. Automata and formal languages are fundamental concepts in Computer Science. Automata are abstract mathematical machines that are used for representing computational processes. Formal languages, on the other hand, are collections of words that we define from a given alphabet (e.g using a grammar). For example, any device interacting with the outside world, whether a simple program or a complex system, requires formal languages for input and output.

    We will study automata and their relationship with formal languages and grammars.

    • Week 5 - Context-free Grammars

      • After studying this section you should be able to:

        • understand context free grammars (CFGs), derivations and parse trees
        • recognise if words belong to the language of a given CFG and, if so, construct derivations
        • explain how to find the yield of a parse tree
        • demonstrate how to draw a parse tree for a given derivation
        • explain what is ambiguity in CFGs
        • give a CFG based on a written description of a language
